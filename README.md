VotoConsciente 2026

O VotoConsciente 2026 é uma plataforma acadêmica desenvolvida para auxiliar o eleitor brasileiro a navegar pelo cenário político das eleições de 2026. Em um momento onde a informação precisa ser acessível e organizada, este projeto busca democratizar o conhecimento sobre as diretrizes e propostas de candidatos a cargos eletivos.

[Acesse o VotoConsciente 2026](https://projetodevweb-eleicoes.web.app/index.html)

🗳️ Sobre a Motivação
Inspirado pela Eleições Presidenciais de 2026, o projeto nasceu da vontade de criar um espaço onde as propostas políticas não ficassem dispersas. A plataforma permite que o cidadão filtre, compare e salve diretrizes, tornando o processo de escolha mais consciente, fundamentado e menos sujeito à desinformação.

🚀 Funcionalidades Principais
Navegação Temática: Filtre propostas por áreas críticas como Saúde, Educação, Segurança e Infraestrutura.

Filtros Avançados: Pesquisa dinâmica por Cargo, Região (Escopo) e Palavras-chave.

Curadoria do Eleitor: Sistema de "Caderninho", onde o usuário logado pode favoritar suas diretrizes preferidas para consulta rápida.

Gestão Administrativa: Área restrita para administradores cadastrarem, editarem e excluírem propostas, mantendo a base de dados atualizada.

Segurança e Autenticação: Sistema de registro e login integrado para personalizar a experiência do usuário.

🛠️ Tecnologias Utilizadas
Front-end: HTML5, CSS3 e JavaScript.

Back-end & Banco de Dados: Firebase (Firestore Database para armazenamento e Firebase Authentication para controle de acessos).

Segurança: Regras de segurança personalizadas no Firestore para proteger os dados dos usuários e restringir acessos administrativos.

💻 Como Rodar o Projeto
Clone este repositório:

git clone https://github.com/felipes08/ProjetoDevWeb_Eleicoes.git
Abra a pasta do projeto em seu editor (VS Code).

O projeto utiliza Firebase. Certifique-se de configurar as credenciais do seu projeto no arquivo main.js.

Utilize uma extensão como o Live Server no VS Code para abrir o arquivo index.html.
